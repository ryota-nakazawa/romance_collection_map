# Romance Collection Map

人物画像をアップロードして、恋愛バラエティ番組風の相関図を作るための静的Webアプリです。

## 使い方

1. `index.html` をブラウザで開く
2. 「人物画像を追加」から画像を選ぶ
3. 人物名を編集する
4. キャンバス上の人物カードをドラッグして配置する
5. From / To / 種類を選び、「関係を追加」で矢印を引く
6. 必要に応じて「PNGを書き出す」から画像化する

## ローカル起動

静的ファイルだけで動きます。ローカルサーバーで開く場合:

```bash
python3 -m http.server 4173
```

その後、ブラウザで `http://localhost:4173` を開きます。

## 主な機能

- 人物画像の追加
- 人物名の編集
- 人物カードのドラッグ配置
- 関係線の追加
- 関係ラベルの編集
- PNG書き出し
- ダミー人物画像
